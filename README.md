# RosterApp - Gestor de Turnos de Empleados

## 📌 Presentación del Proyecto
RosterApp es una Aplicación Web (SPA - Single Page Application) diseñada para digitalizar y centralizar la gestión de cuadrantes de turnos de trabajo en empresas (específicamente orientada a tiendas). El objetivo es eliminar métodos manuales ineficientes, permitiendo a los administradores planificar horarios de forma visual y a los empleados consultar sus turnos desde cualquier dispositivo asegurando la privacidad de sus datos.

## 📂 Estructura del Repositorio
El proyecto contiene la siguiente estructura principal:
* `src/`: Código fuente de la aplicación.
* `docs/`: Documentación del proyecto.
  * `Manual_Usuario.md`: Guía de uso de la aplicación.
  * `Manual_Tecnico.md`: Arquitectura, base de datos y requisitos.
  * `Manual_Despliegue.md`: Instrucciones para ejecutar el aplicativo.
  * `Manual_Proyecto.md`: Resumen de los hitos alcanzados en los dos trimestres.

## 🛠️ Tecnologías Utilizadas
* **Frontend:** React, HTML5 y Tailwind CSS v4 para un diseño ágil y moderno.
* **Backend / Base de Datos:** Supabase (PostgreSQL).
