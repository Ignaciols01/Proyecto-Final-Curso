Este documento detalla los pasos técnicos para descargar, configurar y ejecutar el código fuente de RosterApp en un entorno de desarrollo local.

3.1. Requisitos Previos

El equipo anfitrión debe tener instalados los siguientes componentes:

  • Node.js (Versión LTS recomendada).
  • Git (Para el control de versiones).
  • Editor de código (Recomendado Visual Studio Code).

3.2. Clonación e Instalación

Abra su terminal y clone el repositorio oficial:

git clone [URL_DEL_REPOSITORIO]

Navegue a la carpeta del proyecto e instale las dependencias de React y Tailwind CSS:

cd RosterApp
npm install

3.3. Configuración de Entorno

Cree un archivo llamado .env en la raíz del proyecto para conectar la base de datos de Supabase e inserte sus credenciales:

VITE_SUPABASE_URL=https://[TU_PROYECTO].supabase.co
VITE_SUPABASE_ANON_KEY=[TU_CLAVE_API]

3.4. Ejecución del Servidor Local

Levante el servidor de desarrollo ejecutando el siguiente comando:

npm run dev

La terminal indicará que el aplicativo estará disponible y listo para probarse en http://localhost:5173/.
